Zomato Restaurants Data Analysis
Project Overview

This project focuses on analyzing restaurant data from Zomato using Power BI to discover insights about restaurant ratings, cuisines, online delivery services, table booking availability, cost analysis, and geographical distribution of restaurants.

The dashboard helps users understand customer preferences, restaurant performance, and market trends through interactive visualizations and business intelligence techniques.

Objective

The main objective of this project is to:

Analyze restaurant data from multiple cities and countries
Identify highly rated restaurants and popular cuisines
Study the impact of online delivery and table booking services
Compare restaurant costs and customer ratings
Visualize restaurant distribution geographically
Generate business insights for decision-making
Tools & Technologies Used
Power BI – Data visualization and dashboard creation
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – Calculated measures and KPIs
Excel/CSV Dataset – Source data
Map Visualizations – Geographic analysis

Dashboard Insights

Some important insights derived from the project include:

Restaurants with online delivery generally receive more customer engagement.
Certain cuisines dominate specific cities and regions.
Higher-rated restaurants tend to receive more votes.
Premium restaurants often provide table booking facilities.
Restaurant density is higher in metropolitan areas.
Data Cleaning & Transformation

The following preprocessing steps were performed:

Removed null and duplicate values
Split multiple cuisines into separate columns
Formatted date and numeric values
Created relationships between tables
Generated calculated measures using DAX
